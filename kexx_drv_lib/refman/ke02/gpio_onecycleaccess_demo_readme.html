Example description

This example provides a demonstration of Fast GPIO access. Toggle GPIO PTB2 via Fast GPIO register, the toggle frequency can be as fast as 10MHz, while the core clock (not the bus clock) is 20MHz, indicating the Fast GPIO access in a single cycle. NOTE that the optimization option must be "level high" in C compile configuration.

Code configuration

Make sure the following macros are defined in ke02_config.h:
USE_FEE and
EXT_CLK_FREQ_KHZ macro is defined to 10000.

Directory contents

  • platforms/ke02_config.h Library and board Configuration file
  • projects/ke02/isr.h Interrupt handler installation
  • projects/ke02/GPIO_OneCycleAccess_demo.c Main program

Hardware environment

  • Board: freedom FRDM-KE02Z, revA

How to use it

In order to make the program work, you must do the following :

  • Rebuild all project files and load demo code into the device
  • Measure output frequency on PTB2 (J2 Pin12).
  • The frequency should be 10 MHz.
