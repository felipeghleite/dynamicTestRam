Field Documentation

uint8_t bCounterOverflow

1: overflow bit is set, 0: rno overflow

uint8_t bRegisterReady

1: register has been updated, 0: no up-to-date

uint8_t bSoftReset

1: reset PWT, 0: no action

uint8_t bOverflowIntEn

1: enable overflow interrupt, 0: disable overflow interrupt

uint8_t bReadyIntEn

1: enable data ready interrupt, 0: disable data ready interrupt

uint8_t bPWTIntEn

1: enable PWT module interrupt, 0: disable PWT module interrupt

uint8_t bPWTEn

1: enable PWT module, 0: disable PWT module

uint8_t bClockPrescaler

prescaler value is from 1 to 128

uint8_t bEdgeSensitivity

select edge triggers

uint8_t bPinSelect

input pin select

uint8_t bClockSource

1: PWT clock source is alternative clock, 0: PWT clock source is bus clock
