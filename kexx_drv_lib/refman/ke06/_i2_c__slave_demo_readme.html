Example description

This example provides a template for I2C slave mode. It receive frame and show it on terminal. It also provide a template for how to initialize I2C module. It need to connect the other board programming I2C master code. also can enable the MACRO I2C0_4WIRE_ENABLE, so that user chose the bus voltage level they want and the current he need for strong bus noise immunity.

Code configuration

Make sure the following macros are defined in ke06_config.h:
USE_FEE and
EXT_CLK_FREQ_KHZ macro is defined to 8000(8Mhz).

Directory contents

  • platforms/ke06_config.h library and board Configuration file
  • projects/ke06/isr.h Interrupt handler installation
  • projects/ke06/I2C_Slave_demo.c Main program
  • projects/ke06/i2c_app.c I2C application drivers
  • projects/ke06/i2c_app.h header file of I2C application drivers

Hardware environment

  • Board: freedom FRDM-KE06Z, revA
    • Requirement for demo:
      1. connect two boards with J2 18-18(SDA),20-20(SCL), 14-14(GND)
      2. populate the R47 and R39.

How to use it

In order to make the program work, you must do the following:

  • Rebuild all project files and load demo code into the device
  • Connect J6 USB port to the computer that will find serial terminal port
  • Open the serial terminal
  • Configure the serial terminal to protocol: 8-N-1, 115200bps, and flow control: none
  • Run the example
  • Type any character which will echo to the terminal window
