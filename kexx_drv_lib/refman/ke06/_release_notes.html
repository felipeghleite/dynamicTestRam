Release Notes

Freescale Kinetis KE Peripheral Drivers are bare-metal drivers supporting all Kinetis E seriese devices. It will be released incrementally.

Version Date Description
KE_DRV_1.2.1 2014-3-5 Added Keil support for KE06, support KE02, KE04 and KE06.
KE_DRV_1.1.1 2013-12-11 Ported to KE06 for KE06 alpha program only. This version has no Keil support for KE06. Added CAN demo and updated IIC demo with 4-wire support.
KE_DRV_1.1.0 2013-9-27 Ported to KE04 for KE04 alpha program only. Added Bitband demo.
KE_DRV_1.0.1 2013-8-30 Updated release for 1.0 silicon with bug fixes. See Bug Fixes: section for detail. This version supports KE02 only.
KE_DRV_1.0.0 2013-6-25 Initial release for 1.0 silicon. All sample code have been tested on FRDM-KE02Z Rev A. This version supports KE02 only.

Toolchain Supported:

This driver library supports IAR EWARM version 6.7x or newer,CodeWarrior for MCU 10.5 or newer, KEIL uVision 5.0 or newer.

Bug Fixes:

ENGR00277650: updated PIT period in PIT demo to match 20MHz bus clock
ENGR00272183: fixed some build warnings for Keil
ENGR00277653: updated the comment in ICS mode switch
ENGR00277652: fixed I2C_Slave_demo printf issue
ENGR00273074: fixed some comment issues in demos

Known Issues and Workarounds:

Issue ID Description
ENGR00273346 SPI_SetBaudRate may not set the exact bit rate as expected.
Workaround: update pSPI->BR register to the expected.
ENGR00272184 There are many already open files in default project for IAR.
Workaround: close these files in IAR manually.
ENGR00301726 If using printf function and only use '
' as new line in KEIL,
then the printed message is not aligned on the terminal.
Workaround: configure your terminal to add CR when a New-Line is received.
