Q: Do I need install service pack for CodeWarrior Development Studio for Microcontrollers v10.5?
A: Yes, the KE04/KE06 service pack need to be installed, which can be downloaded from Freescale website.


Q: How to I make IAR 6.7x to work with KE devices?
A: A service pack must be installed for KE04 and KE06, which can be downloaded from Freescale website.


Q: Do I need install service pack for Keil uVision5 V5.01?
A: Yes, the Freescale Kinetis OSJTAG/OpenSDA driver V1.16 or newer must be installed, which can be downloaded from Keil website.
Please follow steps below:

  1. Navigate to the Freescale Kinetis OSJTAG Drivers download page Keil website.
  2. At the Freescale Kinetis OSJTAG Drivers download page, click the FSLKINETISDRIVERSV116.EXE link and follow the onscreen instructions to download this file to the computer.
  3. Navigate to the location of the saved file, double click the executable, and follow the onscreen instructions for installing these files.
