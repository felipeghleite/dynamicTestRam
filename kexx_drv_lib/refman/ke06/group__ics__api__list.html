Macro Definition Documentation

#define ICS_SwitchMode (   CurMode,
  NewMode,
  clkFreq 
)

The clock mode macros are as follows: FEI, FBI, FEE, FBE, FBILP, FBELP, FEE_OSC, FBE_OSC where FEE_OSC, FBE_OSC can not be used as current mode. The valid combinationS of <CurMode, NewMode> pair are as follows: <FEI,FEE>, <FEI,FBI>, <FEI,FBE>, <FEI,FBE_OSC>, <FEI,FEE_OSC>, <FEE,FEI>, <FEE,FBI>, <FEE,FBE>, <FBI,FBE>, <FBI,FEE>, <FBI,FBILP>, <FBI,FEI>, <FBE,FBI>, <FBE,FEE>, <FBE,FEI>, <FBE,FBELP>, <FBELP,FBE>, <FBILP,FBI>.
